# Watson Conversation training toolchain

### Continuously deliver a Watson Conversation application with initial conversation training

This Watson Conversation application uses Node.js and includes a DevOps toolchain that is preconfigured for initial conversation training, Slack notifications, continuous delivery, source control, issue tracking, and online editing.

### To get started, click **Create toolchain**.

[![Deploy To Bluemix](https://console.bluemix.net/devops/graphics/create_toolchain_button.png)](https://console.bluemix.net/devops/setup/deploy/?repository=https%3A//github.com/open-toolchain/watson-training-toolchain)
